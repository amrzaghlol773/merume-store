# Product Images Structure

Place product images in one of these folders:

- /products/men
- /products/women
- /products/candles
- /products/misc

## File naming suggestions

Use simple lowercase names with dashes:

- storm-50ml.jpg
- royal-vetiver-main.jpg
- arabic-wood-100ml.webp

Avoid spaces in file names.

## How to use in admin panel

In image URL fields, use paths like:

- /products/men/storm-50ml.jpg
- /products/women/royal-vetiver-main.jpg
- /products/candles/midnight-oud-1.jpg

You can also use full external URLs, but local paths in /public are recommended for speed and control.
